# CI/CD Pipeline Final Project

## Project Name

CI/CD Pipeline using GitHub Actions, Tekton and OpenShift

## Description

This project demonstrates a complete CI/CD pipeline implementation using GitHub Actions for automation, Tekton for task orchestration, and OpenShift for deployment.

## Pipeline Stages

* Linting using flake8
* Running unit tests using nose
* Tekton cleanup task
* Tekton test task
* OpenShift pipeline deployment

## Tools Used

* GitHub Actions
* Tekton Pipelines
* OpenShift
* Python
